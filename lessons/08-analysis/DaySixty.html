Section 1

Metric Definition

Metric: d: X × X → [0,∞) satisfying:
1. d(x,y) = 0 ⟺ x = y (identity) 2. d(x,y) = d(y,x) (symmetry) 3. d(x,z) ≤ d(x,y) + d(y,z) (triangle inequality)
Section 2

Common Metrics

EUCLIDEAN: d(x,y) = √Σ(xᵢ - yᵢ)² TAXICAB (L¹): d(x,y) = Σ|xᵢ - yᵢ| MAX (L∞): d(x,y) = max|xᵢ - yᵢ| DISCRETE: d(x,y) = 0 if x=y, else 1
Section 3

Metric Topology

OPEN BALL: B(x,r) = {y : d(x,y) < r} OPEN SET: Union of open balls CLOSED SET: Complement of open set CONVERGENCE: xₙ → x iff d(xₙ, x) → 0
