Section 1

Limit Definition

Epsilon-Delta Definition:
lim(x→a) f(x) = L means: ∀ε > 0, ∃δ > 0 such that: 0 < |x - a| < δ ⟹ |f(x) - L| < ε "For every epsilon, there exists a delta..."
Intuition: We can make f(x) arbitrarily close to L by taking x sufficiently close to a.
Section 2

Continuity

f is continuous at a iff: ∀ε > 0, ∃δ > 0 such that: |x - a| < δ ⟹ |f(x) - f(a)| < ε Equivalently: lim(x→a) f(x)=f(a)
Section 3

Proof Technique

TO PROVE lim(x→a) f(x) = L: 1. Let ε > 0 be given 2. Find δ in terms of ε (scratch work) 3. Show: if 0 < |x-a| < δ, then |f(x)-L| < ε 4. Conclude the limit exists
