Section 1

Sequence Convergence

(aₙ) → L means: ∀ε > 0, ∃N such that n > N ⟹ |aₙ - L| < ε BOUNDED + MONOTONE ⟹ CONVERGENT Cauchy sequence: |aₘ - aₙ| < ε for m,n> N In ℝ: Cauchy ⟺ Convergent
Section 2

Series Convergence

Σaₙ converges iff partial sums Sₙ = Σ(k=1 to n) aₖ converge NECESSARY: aₙ → 0 (but not sufficient!) Example: Σ1/n diverges even though 1/n → 0 ABSOLUTE CONVERGENCE: Σ|aₙ| converges ⟹ Σaₙ converges
Section 3

Convergence Tests

COMPARISON: 0 ≤ aₙ ≤ bₙ Σbₙ converges ⟹ Σaₙ converges RATIO TEST: L = lim |aₙ₊₁/aₙ| L < 1 ⟹ converges, L> 1 ⟹ diverges ROOT TEST: L = lim ⁿ√|aₙ| L < 1 ⟹ converges, L> 1 ⟹ diverges INTEGRAL TEST: f decreasing, f(n) = aₙ Σaₙ converges ⟺ ∫f converges ALTERNATING SERIES: (-1)ⁿaₙ, aₙ↓0 ⟹ converges
