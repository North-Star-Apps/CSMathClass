Section 1

Completeness

Complete: Every Cauchy sequence converges.
ℝ is complete (fundamental!) ℚ is NOT complete (e.g., √2 approximations) COMPLETENESS AXIOM: Every nonempty bounded subset of ℝ has a supremum
Section 2

Compactness

DEFINITIONS (equivalent in ℝⁿ): 1. Every open cover has finite subcover 2. Every sequence has convergent subsequence 3. Closed and bounded (Heine-Borel) KEY THEOREM: Continuous image of compact set is compact
Section 3

Uniform Continuity

UNIFORM CONTINUITY: ∀ε > 0, ∃δ > 0: |x-y| < δ ⟹ |f(x)-f(y)| < ε vs POINTWISE CONTINUITY: ∀ε> 0, ∀a, ∃δ(a,ε): |x-a| < δ ⟹ |f(x)-f(a)| < ε KEY: δ doesn't depend on point! THEOREM: Continuous on compact ⟹ uniformly continuous
