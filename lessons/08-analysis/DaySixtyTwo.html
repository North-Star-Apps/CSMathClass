Section 1

Modes of Convergence

POINTWISE: fₙ(x) → f(x) for each x ∀x, ∀ε > 0, ∃N(x,ε): n > N ⟹ |fₙ(x) - f(x)| < ε UNIFORM: sup|fₙ(x) - f(x)| → 0 ∀ε> 0, ∃N(ε): n > N ⟹ sup_x|fₙ(x) - f(x)| < ε (N doesn't depend on x!) Lᵖ CONVERGENCE: ‖fₙ - f‖ₚ → 0 ALMOST EVERYWHERE: fₙ(x) → f(x) except on measure 0
Section 2

Implications

UNIFORM ⟹ POINTWISE (but not conversely!) KEY THEOREM: Uniform limit of continuous functions is continuous (Pointwise limit may not be!) Lᵖ convergence ⟹ subsequence converges a.e.
Counterexample: fₙ(x) = xⁿ on [0,1] converges pointwise to discontinuous limit!
