Section 1

Fourier Series

Joseph Fourier famously claimed that any periodic function can be represented as an weighted sum of sines and cosines:

$$ f(x) = \frac{a_0}{2} + \sum_{n=1}^\infty \left[ a_n \cos(nx) + b_n \sin(nx) \right] $$

This is exactly like projecting a vector onto an orthogonal basis $\{\sin(nx), \cos(nx)\}$. We are shifting from the **Time Domain** to the **Frequency Domain**.

Section 2

The Complex Exponential Form

Using Euler's formula $e^{ix} = \cos(x) + i\sin(x)$, we can write the series more compactly:

$$ f(x) = \sum_{n=-\infty}^\infty c_n e^{inx} $$

Where $c_n$ are complex coefficients. This form is easier to manipulate algebraically and is the standard in engineering and CS.

Section 3

The Continuous Fourier Transform

What if the function isn't periodic? We let the period go to infinity. The sum becomes an integral:

$$ \hat{f}(\xi) = \int_{-\infty}^\infty f(x) e^{-2\pi i x \xi} \, dx $$

This "unwraps" the function into a continuous spectrum of frequencies $\xi$.

Section 4

Core Properties

  • **Linearity:** $\mathcal{F}(af + bg) = a\hat{f} + b\hat{g}$.
  • **Convolution Theorem:** $\mathcal{F}(f * g) = \mathcal{F}(f) \cdot \mathcal{F}(g)$. Convolution in time is multiplication in frequency! This is why filters (blur, sharpen) are so fast to apply.
  • **Plancherel Theorem:** $\int |f|^2 = \int |\hat{f}|^2$. The "energy" (entropy/information) is preserved across transforms.
Section 5

The Uncertainty Principle

In Analysis, the Uncertainty Principle states that a function and its Fourier transform cannot both be "sharply concentrated."

If $f(x)$ is a narrow spike (localized in time), $\hat{f}(\xi)$ must be broad (spread across all frequencies). This is the mathematical reason you can't know a particle's position and momentum perfectly at the same time!

Section 6

Python: Fast Fourier Transform (FFT)

Using the FFT to remove noise from a recorded signal.

Python Signal Denoising 
import numpy as np
import matplotlib.pyplot as plt

# Generate a clean signal (sum of two sines)
t = np.linspace(0, 1, 1000)
clean = np.sin(2 * np.pi * 50 * t) + np.sin(2 * np.pi * 120 * t)

# Add Gaussian noise
white_noise = 2.5 * np.random.randn(len(t))
signal = clean + white_noise

# Compute FFT
fhat = np.fft.fft(signal)
psd = fhat * np.conj(fhat) / len(t) # Power spectrum
freq = (1 / t[1]) * np.arange(len(t)) / len(t)

# Filter out low-power noise
indices = psd > 100
fhat_clean = fhat * indices # Zero out noise
denoised = np.fft.ifft(fhat_clean)

# Plotting
plt.plot(t, signal, color='gray', alpha=0.5, label='Noisy')
plt.plot(t, denoised, color='blue', label='Denoised')
plt.legend()
plt.show()
