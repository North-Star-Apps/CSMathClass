Section 1

The Limit of "Boxes"

In Calculus, we measured area by fitting thin boxes under a curve. This works for smooth functions, but fails for "pathological" ones.

Consider the **Dirichlet Function**: $f(x) = 1$ if $x$ is rational, and $0$ if $x$ is irrational. If you try to integrate this from 0 to 1 with Riemann boxes, you'll find that the "area" is simultaneously 1 and 0! We need a better way.

The Analogy

⚖️ Simple: Counting vs Weighing. Riemann integration is like counting coins by trying to stack them into neat towers. Measure theory is like weighing the coins. Even if the coins are scattered randomly, the weight (measure) remains a valid total.

Section 2

Outer Measure ($m^*$)

We start by covering a set $E$ with a countable collection of open intervals $I_n$. The **Outer Measure** of $E$ is the infimum of the sums of the lengths of these intervals:

$$ m^*(E) = \inf \left\{ \sum \ell(I_n) : E \subset \bigcup I_n \right\} $$

This gives us an "upper bound" on the size of any set. But not all sets are "measurable" (behaving nicely with addition).

Section 3

Sigma-Algebras ($\mathcal{F}$)

A collection of sets is a **$\sigma$-algebra** if it contains the empty set, is closed under complements, and is closed under countable unions.

In Probability, this is the collection of all "events" we can assign a probability to. In Analysis, these are the "measurable sets."

Section 4

The Lebesgue Measure ($\mu$)

Henri Lebesgue generalized the concept of length. For a measurable set $E \subset \mathbb{R}$, we define $\mu(E)$ such that it matches the length for intervals, but also gives us the correct "size" for weird sets.

  • The measure of all rationals in $[0, 1]$ is **0**.
  • The measure of all irrationals in $[0, 1]$ is **1**.

Because there are "so many more" irrationals than rationals, the "weight" of the set is dominated by them.

Section 5

Sets of Measure Zero

A set $E$ has **Measure Zero** if you can cover it with intervals of arbitrarily small total length.

If something is true "almost everywhere" ($a.e.$), it means it's true except on a set of measure zero. In Probability, this is "almost surely."

Section 6

Python: The Cantor Set

The Cantor Set is created by repeatedly removing the middle third of intervals. It is uncountably infinite (like the reals) but has **measure zero**! It's "infinite dust."

Python Cantor Construction 
def cantor_measure(iterations):
    # Start with length 1
    length = 1.0
    for i in range(iterations):
        # Every step removes the middle 1/3, keeping 2/3
        length *= (2/3)
        print(f"Iteration {i+1}: Remaining Measure = {length:.6f}")
    return length

print("Calculating the measure of the Cantor Set as iterations -> inf:")
cantor_measure(20)

# Mathematically, the measure is (2/3)^n which goes to 0 as n -> inf.
