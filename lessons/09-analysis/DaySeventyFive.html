Section 1

Functions as Vectors

In Linear Algebra, we worked with vectors in $\mathbb{R}^n$. Functional Analysis extends this to "infinite" dimensions where each **function** is a point in the space.

The **$L^p$ Spaces** are sets of functions where the $p$-th power of the absolute value is integrable:

$$ \|f\|_p = \left( \int |f|^p \, d\mu \right)^{1/p} < \infty $$
  • **$L^1$:** Functions that are absolutely integrable (Probability density functions).
  • **$L^2$:** The most important one! Functions that have a finite "energy."
  • **$L^\infty$:** Functions that are bounded almost everywhere.
Section 2

The Inner Product of Functions

In $L^2$, we can define an **Inner Product** similar to the dot product:

$$ \langle f, g \rangle = \int f(x) \overline{g(x)} \, d\mu $$

This allows us to speak about the **Angle** between two functions and the **Length** (Norm) of a function.

Section 3

Hilbert Spaces

A **Hilbert Space** is an inner product space that is **complete** (every Cauchy sequence converges). This is where all the best tools of Linear Algebra (eigenvalues, projections, basis) work in infinite dimensions.

The CS Connection: Kernels

In Support Vector Machines (SVMs), the "Kernel Trick" implicitly maps data into a high-dimensional (or even infinite-dimensional) Hilbert Space where the data becomes linearly separable. This is known as **Reproducing Kernel Hilbert Spaces (RKHS)**.

Section 4

Infinite Orthogonality

Just as we used $\{ \mathbf{e}_1, \mathbf{e}_2 \}$ for $\mathbb{R}^2$, we can find a **basis** for $L^2$.

Example: The set of sines and cosines $\{\sin(nx), \cos(nx)\}$ is an orthogonal basis for periodic functions (the foundation of the Fourier Transform).

Convergence Matters

In infinite dimensions, we have to be careful about what it means for a sum to equal a function. Usually, we mean convergence "in the $L^2$ sense," not necessarily point-by-point.

Section 5

Linear Operators

Matrices transform vectors. In Functional Analysis, **Linear Operators** transform functions ($T: X \to Y$).

**The Derivative** is a linear operator: $D(f+g) = Df + Dg$.
**The Integral** is a linear operator.

The "eigenvectors" of these operators are called **eigenfunctions**. This is the core of solving differential equations and understanding the Schrödinger equation in physics.

Section 6

Python: Projecting onto Legendre Polynomials

We often approximate functions using a finite number of orthogonal basis functions.

Python Function Approximation 
import numpy as np
from scipy.special import legendre
import matplotlib.pyplot as plt

def f(x):
    # Function to approximate
    return np.exp(x)

x = np.linspace(-1, 1, 100)
y_actual = f(x)

# Approximate using first 4 Legendre polynomials
# (They are orthogonal on [-1, 1])
approx = np.zeros_like(x)
for n in range(4):
    Pn = legendre(n)
    # Compute the projection (inner product)
    #  / 
    poly_vals = Pn(x)
    coeff = np.sum(f(x) * poly_vals) / np.sum(poly_vals**2)
    approx += coeff * poly_vals

plt.plot(x, y_actual, label='Actual exp(x)')
plt.plot(x, approx, '--', label='Poly Apprx (n=4)')
plt.legend()
plt.show()
