Section 1

The Masterwork: Capstone Project

Mathematics is a tool for building. To graduate, you must submit a **Capstone Project** that combines at least three major branches of this curriculum. Examples include:

  • **A Neural Network from Scratch:** Using Linear Algebra (Matrices), Calculus (Backprop), and Probability (Initialization).
  • **A Search Engine:** Using Discrete Math (Graph Traversal), Information Theory (Entropy), and Data Structures (B-Trees/Hash Maps).
  • **A Graphics Engine:** Using 3D Linear Algebra, Quaternions, and Calculus (Shaders/Integration).
