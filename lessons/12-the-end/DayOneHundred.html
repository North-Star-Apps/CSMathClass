Section 2

The Frontier (Unsolved Problems)

You are now equipped to understand the cutting edge. Here lie the dragons.

P vs NP

Can every problem whose solution can be quickly verified also be quickly solved?
If P = NP, then cryptography is broken, we can cure cancer (protein folding), and perfect optimization is possible.
Most believe P != NP, but we cannot prove it.

üß©
Analogy: The Jigsaw Puzzle
Verification (P): If I show you a completed puzzle, you can instantly verify it's correct (edges match, image looks right).
Solution (NP): If I dump the pieces on the table, it takes hours to solve.
P vs NP: Asks "Is there a secret trick to solve the puzzle as fast as checking it?"

The Riemann Hypothesis

All non-trivial zeros of the Riemann Zeta function have real part 1/2.
This connects discrete primes to continuous analysis. It is the holy grail of Number Theory.

Section 5

Complexity from Simplicity

We end with Conway's Game of Life. From simple rules, infinite complexity emerges. This is the lesson of Computer Science: 0s and 1s can create Universes.

üêú
Analogy: The Ant Colony
A single ant follows very simple chemical rules (follow trail, find food). It has no concept of "architecture."
Yet, millions of ants together build climate-controlled skyscrapers, farm fungus, and wage war.
Emergence: Complex behavior arising from simple parts.
Python Conway's Game of Life (Minimal) 
import time
import os

def game_of_life(size=10, generations=5):
    # Glider pattern
    grid = set([(1,2), (2,3), (3,1), (3,2), (3,3)])
    
    for gen in range(generations):
        # Calc neighbors
        neighbors = {}
        for (x,y) in grid:
            for dx in [-1,0,1]:
                for dy in [-1,0,1]:
                    if dx==0 and dy==0: continue
                    neighbors[(x+dx, y+dy)] = neighbors.get((x+dx, y+dy), 0) + 1
        
        # Apply rules
        new_grid = set()
        for cell, count in neighbors.items():
            if count == 3 or (count == 2 and cell in grid):
                new_grid.add(cell)
        grid = new_grid
        
        # Render
        print(f"--- Gen {gen} ---")
        for x in range(size):
            line = ""
            for y in range(size):
                line += "O" if (x,y) in grid else "."
            print(line)
        print()

game_of_life()
