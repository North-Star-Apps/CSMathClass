The Enemy: Ebbinghaus

Hermann Ebbinghaus discovered that human memory decays exponentially. Without review, you forget 50% of what you learned within 24 hours, and 80% within a month.

$$ R = e^{-\frac{t}{S}} $$

Where $R$ is retentivity, $t$ is time, and $S$ is strength of memory. To master math, we must artificially increase $S$.

üìâ Simple: The Leaky Bucket Analogy
Your memory is like a leaky bucket. New knowledge drips in from the top, but slowly leaks out the bottom. Each time you review, you patch the holes a bit‚Äîmaking the bucket hold water longer. Spaced repetition = patching at optimal times!
Section 2

The Two-Pass Strategy

High-performers rarely learn something to 100% on the first try. They use the **Two-Pass Method**:

  • **Pass 1 (Survey):** Read for intuition. Solve "Easy" problems. Don't get stuck on proofs. Mark the "Hard" parts.
  • **Pass 2 (Synthesis):** Re-read the "Hard" parts now that you have the big picture. Solve the rigorous proofs. Re-implement the code from memory.
Section 3

Active Recall

Passive review (re-reading notes) is an illusion of competence. **Active Recall** (testing yourself without looking) actually builds the neural pathways.

If it feels difficult and frustrating, it's working. If it feels smooth and comfortable, you're likely just recognizing patterns, not learning them.

üèãÔ∏è Simple: The Gym Analogy
Re-reading is like watching workout videos. Active recall is actually lifting weights. The struggle IS the learning. No pain, no brain gain!
Section 4

Spaced Repetition (The Algorithm)

Reviewing right before you're about to forget is the "Golden Moment." The Anki/SuperMemo algorithms use a simple scheduling rule: If you get it right, double the interval ($1, 2, 4, 8, 16$ days). If you miss it, reset to 1.

Python: Decay Simulation

Let's simulate how review "resets" the forgetting curve and makes the decay shallower over time.

Python Memory Simulator 
import numpy as np

def simulate_memory(days, reviews=[]):
    memory = 100.0
    half_life = 2.0 # Initial half-life in days
    history = []
    
    for day in range(days):
        if day in reviews:
            memory = 100.0 # Review resets memory
            half_life *= 2.0 # Memory gets 'stronger'
        
        history.append(memory)
        # Exponential decay
        memory *= np.exp(-np.log(2)/half_life)
        
    return history

# No review
passive = simulate_memory(30)
# Review on Day 1, 3, 7, 15
active = simulate_memory(30, reviews=[1, 3, 7, 15])

print(f"Memory after 30 days (No review): {passive[-1]:.1f}%")
print(f"Memory after 30 days (Active):    {active[-1]:.1f}%")
