Matrices are everywhere. Graphics, ML, scientific computing—all built on efficient matrix operations.

Matrix Operations

Addition & scalar multiplication

A + B: element-wise addition (same dimensions required) cA: multiply every element by scalar c [1 2] [5 6] [6 8] [3 4] + [7 8] = [10 12]

Matrix multiplication

Matrix Multiplication
(AB)ᵢⱼ = Σₖ Aᵢₖ Bₖⱼ (row i of A · column j of B)
A is m×n, B is n×p → AB is m×p Key properties: • AB ≠ BA in general (not commutative!) • (AB)C = A(BC) (associative) • A(B+C) = AB + AC (distributive) [1 2] [5 6] [1·5+2·7 1·6+2·8] [19 22] [3 4] [7 8] = [3·5+4·7 3·6+4·8] = [43 50]

Transpose

(Aᵀ)ᵢⱼ = Aⱼᵢ (flip rows and columns) Properties: • (Aᵀ)ᵀ = A • (AB)ᵀ = BᵀAᵀ (note the order reversal!)

Section 2

Gaussian Elimination

Row operations

Three allowed operations (don't change solution set): 1. Swap two rows 2. Multiply a row by nonzero scalar 3. Add multiple of one row to another

Row Echelon Form (REF)

• Leading entry (pivot) of each row is to right of row above • Rows of zeros at bottom [2 1 -1 | 8] [2 1 -1 | 8] [4 3 1 | 12] → [0 1 3 | -4] [2 1 2 | 3] [0 0 5 | -5]

Reduced REF (RREF)

• All pivots are 1 • Each pivot is the only nonzero in its column [1 0 0 | 2] [0 1 0 | 1] ← Solution: x=2, y=1, z=-1 [0 0 1 | -1]
Section 3

Rank

Definition

Rank
rank(A) = # of pivots = dim(column space) = dim(row space)

Key facts

For m×n matrix A: • rank(A) ≤ min(m, n) • rank(A) = rank(Aᵀ) • Full row rank: rank(A) = m • Full column rank: rank(A) = n For Ax = b: • Unique solution ⟺ rank(A) = n and rank(A|b) = rank(A) • No solution ⟺ rank(A|b) > rank(A) • Infinite solutions ⟺ rank(A) < n
Rank-Nullity Theorem: rank(A) + nullity(A) = n (# columns)

Matrix Inverse

Definition

A⁻¹ exists ⟺ A is square and det(A) ≠ 0 AA⁻¹ = A⁻¹A = I Properties: • (A⁻¹)⁻¹ = A • (AB)⁻¹ = B⁻¹A⁻¹ (order reverses!) • (Aᵀ)⁻¹ = (A⁻¹)ᵀ

2×2 formula

[a b]⁻¹ 1 [ d -b] [c d] = ――――― [-c a] ad-bc det = ad-bc (must be nonzero)

Finding inverse via elimination

Augment [A | I], reduce to RREF: [A | I] → [I | A⁻¹]
