Section 1

The "Bowl" Property

A function is **Convex** if a line segment between any two points on its graph lies above or on the graph.

$$ f(tx_1 + (1-t)x_2) \leq tf(x_1) + (1-t)f(x_2) $$

Visually: It's a bowl shape. If you drop a ball anywhere, it will roll to the absolute bottom.

Section 2

Global vs Local

The Holy Grail of optimization: In a **Convex Function**, any local minimum is automatically the **Global Minimum**.

This is why we love Linear Regression and Support Vector Machines—they are convex problems. Deep Learning, however, is notoriously non-convex!

Section 3

The Hessian Matrix ($H$)

While the Gradient tells us the slope, the **Hessian** tells us the **curvature**.

$$ H_{ij} = \frac{\partial^2 f}{\partial x_i \partial x_j} $$

If the Hessian is **Positive Definite** everywhere, the function is strictly convex.

Section 4

Second-Order Taylor

To approximate a function near a point $x$:

$$ f(x + \Delta x) \approx f(x) + \nabla f(x)^T \Delta x + \frac{1}{2} \Delta x^T H(x) \Delta x $$

This is the basis for **Newton's Method**, which takes faster steps than Gradient Descent by using curvature information.

Section 5

Python: Checking Convexity

Checking if a matrix is Positive Definite (using eigenvalues).

Python Convexity Check 
import numpy as np

def is_convex(hessian_matrix):
    # A function is convex if its Hessian is Positive Semi-Definite
    # i.e., all its eigenvalues are >= 0
    eigenvalues = np.linalg.eigvals(hessian_matrix)
    print("Eigenvalues:", eigenvalues)
    return np.all(eigenvalues >= 0)

# Example 1: f(x,y) = x^2 + y^2 (A simple bowl)
# Hessian is [[2, 0], [0, 2]]
H1 = np.array([[2, 0], [0, 2]])
print("H1 Convex?", is_convex(H1))

# Example 2: f(x,y) = x^2 - y^2 (A saddle)
# Hessian is [[2, 0], [0, -2]]
H2 = np.array([[2, 0], [0, -2]])
print("\nH2 Convex?", is_convex(H2))
