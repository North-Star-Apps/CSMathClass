Section 1

Orthogonal Projection

When a vector $\mathbf{b}$ lies outside a subspace $H$, the "closest" vector in $H$ to $\mathbf{b}$ is its **Orthogonal Projection**, $\hat{\mathbf{b}}$.

The "error" vector $\mathbf{b} - \hat{\mathbf{b}}$ is orthogonal (perpendicular) to the subspace.

Section 2

Unsolvable Systems

In real life, $Ax=b$ often has no solution because there is too much noise. (More equations than variables).

Instead of giving up, we find $\hat{x}$ that minimizes the error $||Ax - b||^2$.

This is called the **Least Squares Solution**.

Section 3

The Normal Equation

To find the least squares solution $\hat{x}$, we solve:

$$ A^T A \hat{x} = A^T \mathbf{b} $$

If $A^T A$ is invertible, then:

$$ \hat{x} = (A^T A)^{-1} A^T \mathbf{b} $$

The matrix $(A^T A)^{-1} A^T$ is called the **Pseudo-Inverse** ($A^\dagger$).

Section 4

Linear Regression

Fitting a line $y = mx + c$ to data points is literally just solving the Normal Equation.

We construct matrix $A$ with a column of $x$-values and a column of 1s (for the intercept).

Section 5

Python: Regression from Scratch

Using the Normal Equation to fit a line, comparing to `numpy.polyfit`.

Python Least Squares 
import numpy as np
import matplotlib.pyplot as plt

# 1. Generate Data with Noise
X_vals = np.array([1, 2, 3, 4, 5])
y_vals = np.array([2.1, 3.9, 6.2, 8.1, 9.8]) # Approx y = 2x

# 2. Build Matrix A (Add column of 1s for intercept)
# Column 1: X values, Column 2: 1s
A = np.column_stack((X_vals, np.ones_like(X_vals)))
print("Design Matrix A:\n", A)

# 3. Solve Normal Equation: (A.T @ A) * x = A.T @ y
# x_hat = [slope, intercept]
x_hat = np.linalg.inv(A.T @ A) @ A.T @ y_vals

slope, intercept = x_hat
print(f"Slope (m): {slope:.2f}, Intercept (c): {intercept:.2f}")

# 4. Verify with numpy
m_np, c_np = np.polyfit(X_vals, y_vals, 1)
print(f"Numpy Polyfit: m={m_np:.2f}, c={c_np:.2f}")

# 5. Plot
plt.scatter(X_vals, y_vals, color='red', label='Data')
plt.plot(X_vals, slope*X_vals + intercept, color='blue', label='Least Squares Line')
plt.legend()
plt.show()
