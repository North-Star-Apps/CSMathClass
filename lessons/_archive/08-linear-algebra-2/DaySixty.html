Section 1

The Basis

A set of vectors $\{\beta_1, ..., \beta_n\}$ is a Basis for a subspace $H$ if:

  1. They are Linearly Independent (no redundancy).
  2. They Span $H$ (cover everything).

Analogy: The minimal ingredients needed to bake the cake. Standard basis for $\mathbb{R}^2$ is $\mathbf{i} = [1, 0]$ and $\mathbf{j} = [0, 1]$.

Section 2

Dimension

The **Dimension** of a nonzero subspace $H$, denoted $\dim H$, is the number of vectors in any basis for $H$.

  • $\dim \mathbb{R}^3 = 3$.
  • A line through the origin in $\mathbb{R}^3$ has dimension 1.
  • A plane through the origin in $\mathbb{R}^3$ has dimension 2.
Section 3

Column Space & Null Space

For an $m \times n$ matrix $A$:

Column Space ($Col A$)

Span of the columns of $A$.

Subspace of $\mathbb{R}^m$.

Includes all $b$ such that $Ax=b$ has a solution.

Null Space ($Nul A$)

Set of all $x$ such that $Ax = \mathbf{0}$.

Subspace of $\mathbb{R}^n$.

Section 4

Rank-Nullity Theorem

This links the dimensions of the four subspaces. For any $m \times n$ matrix $A$:

$$ \text{Rank}(A) + \dim(\text{Nul } A) = n $$

($n$ = number of columns).
Rank is the number of pivot columns. Nullity is the number of free variables.

Section 5

Python: Finding a Basis

Using SymPy to find the Reduced Row Echelon Form (RREF) to identify pivot columns (Basis columns).

Python RREF & Basis 
import sympy as sp

# 1. Define Matrix (Matrix has dependent columns)
M = sp.Matrix([
    [1, 2, 0, 1],
    [2, 4, 1, 4],
    [3, 6, 3, 9]
])

# 2. Get Reduced Row Echelon Form
# rref returns (matrix, pivot_columns_indices)
rref_matrix, pivots = M.rref()

print("RREF Matrix:")
sp.pprint(rref_matrix)
print("\nPivot Columns Indices:", pivots)

# 3. Extract Basis for Column Space
# The pivot columns of the ORIGINAL matrix form the basis
basis_vectors = [M.col(i) for i in pivots]
print("\nBasis for Col(A):")
print(basis_vectors)

# Notice Column 2 was redundant (2 * Col 1), so it's not a pivot.
