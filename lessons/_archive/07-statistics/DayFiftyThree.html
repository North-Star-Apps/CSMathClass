Section 1

Bernoulli & Binomial

Bernoulli(p)

A single Yes/No trial (e.g., Coin Flip).

  • $P(X=1) = p$
  • $P(X=0) = 1-p$

Binomial(n, p)

Sum of $n$ independent Bernoulli trials.

Probability of getting exactly $k$ successes:

$$ P(X=k) = \binom{n}{k} p^k (1-p)^{n-k} $$

Section 2

Uniform Distribution

All outcomes are equally likely.

  • Discrete: Rolling a die ($P=1/6$).
  • Continuous: Selecting a point on a line segment $[a, b]$. $f(x) = \frac{1}{b-a}$.
Section 3

The Normal (Gaussian)

The Bell Curve. Occurs naturally due to the Central Limit Theorem.

$$ X \sim \mathcal{N}(\mu, \sigma^2) $$

68-95-99.7 Rule:

  • 68% of data within $1\sigma$.
  • 95% of data within $2\sigma$.
  • 99.7% of data within $3\sigma$.
Section 4

Poisson (Rare Events)

Counts events occurring at a constant rate within a fixed interval (e.g., number of emails per hour).

$$ P(X=k) = \frac{e^{-\lambda} \lambda^k}{k!} $$

Parameter $\lambda$ is the average rate.

Section 5

Python: Scipy Stats

Generating distributions and calculating probabilities.

Python Normal Distribution 
from scipy.stats import norm
import numpy as np

# A Normal Dist with mean=0, std=1 (Standard Normal)
dist = norm(loc=0, scale=1)

# 1. PDF (Value of curve at x)
print(f"PDF at 0: {dist.pdf(0):.4f}") # Peak ~0.3989

# 2. CDF (Probability X <= x)
print(f"CDF at 0: {dist.cdf(0)}") # 0.5 (Half the curve)

# 3. Probability between -1 and 1 (1 SD)
prob_1sd = dist.cdf(1) - dist.cdf(-1)
print(f"68% Rule Check: {prob_1sd:.4f}") # 0.6827

# 4. Generate Random Sample
sample = dist.rvs(size=5)
print(f"Random Sample: {sample}")
