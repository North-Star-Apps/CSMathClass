Section 1

The Sample Space ($\Omega$)

The set of all possible outcomes of an experiment.

  • Coin Flip: $\Omega = \{H, T\}$
  • Die Roll: $\Omega = \{1, 2, 3, 4, 5, 6\}$
  • Two Coins: $\Omega = \{HH, HT, TH, TT\}$

An Event ($A$) is any subset of $\Omega$.

$$ P(A) = \frac{|A|}{|\Omega|} \quad \text{(If all outcomes equally likely)} $$
Section 2

Counting Principles

To calculate probabilities, we often need to count huge numbers of possibilities.

Permutations (Order Matters)

Arranging $n$ distinct items.

$$ P(n, k) = \frac{n!}{(n-k)!} $$

Example: 1st, 2nd, 3rd place in a race.

Combinations (Order Doesn't Matter)

Choosing $k$ items from $n$.

$$ C(n, k) = \binom{n}{k} = \frac{n!}{k!(n-k)!} $$

Example: Picking a committee of 3 people.

Section 3

Axioms of Probability

Kolmogorov's Foundation:

  1. $0 \le P(A) \le 1$
  2. $P(\Omega) = 1$
  3. If $A$ and $B$ are disjoint ($A \cap B = \emptyset$): $$ P(A \cup B) = P(A) + P(B) $$

From these, we derive rules like $P(\text{not } A) = 1 - P(A)$.

Section 4

Python: Itertools

Python's `itertools` library is a superpower for counting.

Python Permutations vs Combinations 
import itertools

items = ['A', 'B', 'C']

# 1. Permutations (Order matters)
perms = list(itertools.permutations(items, 2))
print(f"Permutations (2 of 3): {perms}")
# Output: [('A', 'B'), ('A', 'C'), ('B', 'A'), ('B', 'C'), ('C', 'A'), ('C', 'B')]
# count = 3! / (3-2)! = 6 / 1 = 6

# 2. Combinations (Order doesn't matter)
combs = list(itertools.combinations(items, 2))
print(f"Combinations (2 of 3): {combs}")
# Output: [('A', 'B'), ('A', 'C'), ('B', 'C')]
# count = 3! / (2! * 1!) = 6 / 2 = 3

# 3. Cartesian Product (Sample Space of 2 coin flips)
coins = ['H', 'T']
space = list(itertools.product(coins, repeat=2))
print(f"Sample Space: {space}")
# Output: [('H', 'H'), ('H', 'T'), ('T', 'H'), ('T', 'T')]
