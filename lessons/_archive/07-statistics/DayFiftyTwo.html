Section 1

Expectation (The Mean)

The "Center of Mass" of a distribution. Weighted average of all possible values.

$$ E[X] = \sum x \cdot P(X=x) $$
Section 2

Variance & Standard Deviation

Technically: "The expected squared deviation from the mean".

Intuitively: How "spread out" or "risky" the data is.

$$ Var(X) = E[(X - \mu)^2] = E[X^2] - (E[X])^2 $$

Standard Deviation ($\sigma$): $\sqrt{Var(X)}$. This is in the same units as the data (e.g., dollars, meters).

Section 3

Linearity of Expectation

A powerful property that holds even if variables are dependent.

$$ E[A + B] = E[A] + E[B] $$

Example: Expected number of Heads in 100 coin flips = $E[X_1] + ... + E[X_{100}] = 100 \times 0.5 = 50$.

Section 4

Python: Calculating Stats

Using `numpy` to compute mean and variance.

Python Stats Calculations 
import numpy as np

# Data: 5 students' scores
scores = np.array([80, 85, 90, 95, 100])

# 1. Mean (Expectation)
mean = np.mean(scores)
print(f"Mean: {mean}") # 90.0

# 2. Variance
var = np.var(scores)
print(f"Variance: {var}") # 50.0

# 3. Standard Deviation
std = np.std(scores)
print(f"Std Dev: {std:.2f}") # 7.07

# 4. Linearity Check
# Add 5 points to everyone. Mean should increase by 5.
scores_boosted = scores + 5
print(f"New Mean: {np.mean(scores_boosted)}") # 95.0
