Section 1

Conditional Probability

The probability of event $A$, given that event $B$ has already occurred.

$$ P(A|B) = \frac{P(A \cap B)}{P(B)} $$

Independence: If knowing $B$ doesn't change probability of $A$, then $P(A|B) = P(A)$.

Section 2

Bayes' Theorem

The Formula

$$ P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)} $$

  • $P(A)$: Prior (Initial belief)
  • $P(B|A)$: Likelihood (Data given theory)
  • $P(A|B)$: Posterior (Updated belief)
Section 3

Total Probability Theorem

Sometimes finding $P(B)$ directly is hard. We break it down into cases.

If $A_1, A_2 ... A_n$ cover the sample space:

$$ P(B) = \sum P(B|A_i) P(A_i) $$

Python: Monty Hall

A classic counter-intuitive problem. 3 doors. 1 car, 2 goats. You pick a door. Host opens a goat door. Should you switch?

Python Monty Hall Simulation 
import random

def monty_hall_sim(trials=10000):
    stay_wins = 0
    switch_wins = 0
    
    for _ in range(trials):
        # 1. Setup
        doors = [0, 0, 1] # 0=Goat, 1=Car
        random.shuffle(doors)
        
        # 2. Player picks
        choice = random.randint(0, 2)
        
        # 3. Host opens a Goat door (not player's choice)
        available_doors = [i for i in range(3) if i != choice and doors[i] == 0]
        host_opens = random.choice(available_doors)
        
        # 4. Result if Stay
        if doors[choice] == 1:
            stay_wins += 1
            
        # 5. Result if Switch
        remaining_door = [i for i in range(3) if i != choice and i != host_opens][0]
        if doors[remaining_door] == 1:
            switch_wins += 1
            
    print(f"Stay Win Prob: {stay_wins/trials:.3f}")   # ~0.333
    print(f"Switch Win Prob: {switch_wins/trials:.3f}") # ~0.666

monty_hall_sim()

Conclusion: Always switch! Your initial chance was 1/3. The other 2/3 chance is concentrated in the remaining door.
