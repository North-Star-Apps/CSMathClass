Section 1

Point Estimation

Using sample data to calculate a single value (a guess) for an unknown population parameter.

  • Sample Mean ($\bar{x}$): Estimator for Population Mean ($\mu$).
  • Sample Variance ($s^2$): Estimator for Population Variance ($\sigma^2$).

An estimtor gives us a "point", but points are rarely exactly right. We need a range.

Section 2

Confidence Intervals (CI)

A range of values that we are "95% confident" contains the true parameter.

$$ \bar{x} \pm Z \cdot \frac{s}{\sqrt{n}} $$

Meaning: If we repeated this experiment sizes of times, 95% of the calculated intervals would contain the true $\mu$.

Section 3

Standard Error (SE)

The Standard Deviation of the sampling distribution of a statistic.

$$ SE = \frac{\sigma}{\sqrt{n}} \approx \frac{s}{\sqrt{n}} $$

Basically: How much we expect our sample mean to wiggle if we took another sample.

Section 4

The Bootstrap Method

A computational magic trick. We treat our single sample as a "mini-population".

  1. Resample from our data with replacement.
  2. Calculate the statistic (e.g., mean) for this new "bootstrap sample".
  3. Repeat 10,000 times.
  4. The distribution of these bootstrap stats approximates the sampling error!
Section 5

Python: Bootstrapping

Calculating a 95% CI without complex formulas.

Python Bootstrap CI 
import numpy as np

# 1. Original Sample (e.g., heights)
data = np.array([170, 172, 168, 175, 180, 165, 171, 174])
original_mean = np.mean(data)

# 2. Bootstrap Loop
n_iterations = 10000
boot_means = []

for _ in range(n_iterations):
    # Resample with replacement
    sample = np.random.choice(data, size=len(data), replace=True)
    boot_means.append(np.mean(sample))

# 3. Calculate Confidence Interval
# Get 2.5th and 97.5th percentiles
ci_lower = np.percentile(boot_means, 2.5)
ci_upper = np.percentile(boot_means, 97.5)

print(f"Original Mean: {original_mean:.2f}")
print(f"95% CI: [{ci_lower:.2f}, {ci_upper:.2f}]")
