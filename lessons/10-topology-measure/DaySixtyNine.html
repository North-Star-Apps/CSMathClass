Section 1

Compactness

The "Finite Subcover" Definition

Compact: A space X is compact if every open cover of X has a finite subcover.
Why "Finite"? If you cover infinite X with infinite patches, compactness guarantees you only "actually needed" a few of them. This makes infinite spaces behave like finite sets (e.g., Min/Max value theorem works). Heine-Borel: In ℝⁿ, Compact ⟺ Closed and Bounded. (This is FALSE for general metric spaces!)
Section 2

Connectedness

Definitions

Connected: X CANNOT be written as the union of two disjoint, non-empty open sets. (You can't "slice" it open cleanly). Path Connected: For any x, y ∈ X, there is a continuous path f: [0,1] → X from x to y.
Path Connected ⟹ Connected (But Connected ⇏ Path Connected, e.g., Topologist's Sine Curve).
Python Graph Connectivity (Analogy) 
import networkx as nx
import matplotlib.pyplot as plt

G = nx.Graph()
G.add_edges_from([(1, 2), (2, 3)]) # Component 1
G.add_node(4)                    # Component 2

print(f"Is connected? {nx.is_connected(G)}")
# Output: False

# In Topology, "disjoint open sets" are like
# graph components that don't touch.

Separation Axioms

Hausdorff Spaces (T2)

Hausdorff: For any two distinct points x, y, there exist disjoint open sets U, V such that x∈U and y∈V. "You can insulate points from each other." Most nice spaces (manifolds, metric spaces) are Hausdorff. Limits of sequences are unique ONLY in Hausdorff spaces.
