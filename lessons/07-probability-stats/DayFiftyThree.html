Point Estimation

ESTIMATOR: Statistic used to estimate parameter ESTIMATE: Value of estimator for given sample Sample mean X̄ estimates population mean μ Sample variance s² estimates population variance σ² PROPERTIES: • Unbiased: E[θ̂] = θ • Consistent: θ̂ → θ as n → ∞ • Efficient: minimum variance among unbiased
Section 2

Confidence Intervals

95% CI: Interval that contains true parameter 95% of the time.
For mean with known σ: CI = X̄ ± z_(α/2) · σ/√n For mean with unknown σ: CI = X̄ ± t_(α/2, n-1) · s/√n Common z values: 90% CI: z = 1.645 95% CI: z = 1.96 99% CI: z = 2.576
Section 3

Hypothesis Testing

H₀: Null hypothesis (status quo) H₁: Alternative hypothesis PROCEDURE: 1. State H₀ and H₁ 2. Choose significance level α 3. Compute test statistic 4. Find p-value 5. Reject H₀ if p-value < α ERRORS: Type I: Reject H₀ when true (prob=α) Type II: Fail to reject H₀ when false (prob=β)
