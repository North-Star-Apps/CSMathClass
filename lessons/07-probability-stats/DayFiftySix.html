Section 1

Point Estimators

An Estimator $\hat{\theta}$ is a rule (function of the data) used to guess a population parameter $\theta$.

  • Unbiased: $E[\hat{\theta}] = \theta$. On average, the estimator is correct.
  • Efficiency: The estimator with the lowest variance is better.
üéØ Simple: The Sniper Analogy
The "bias" is how far the center of your bullet holes is from the bullseye. The "variance" is how spread out the holes are. A perfect sniper has zero bias and zero variance.
Section 2

Maximum Likelihood (MLE)

MLE is the most common way to find estimators. We ask: "Which parameter value $\theta$ makes the observed data most likely?"

$L(\theta) = P(X_1, X_2, ..., X_n | \theta)$
Find $\theta$ that maximizes $L(\theta)$.
üïµÔ∏è Simple: The Likeliest Suspect
You find a crime scene (data). You have 3 suspects (parameter values). You ask: "If suspect A were the killer, how likely is it that the room would look exactly like this?" The suspect with the highest probability is your MLE.
Section 3

Confidence Intervals

Instead of a single number, we provide a range of values that likely contains the true parameter.

A $95\%$ Confidence Interval means: "If we repeated this experiment 100 times, 95 of the intervals we calculated would contain the true mean."

$$ CI = \bar{X} \pm Z^* \left(\frac{\sigma}{\sqrt{n}}\right) $$
  • 90%: $Z^* = 1.645$
  • 95%: $Z^* = 1.96$
  • 99%: $Z^* = 2.576$
Section 4

Python: Statistical Inference

Calculating Confidence Intervals with `scipy`.

Python Confidence Intervals 
import numpy as np
import scipy.stats as stats

data = [102, 98, 101, 105, 97, 99, 103, 100, 101, 98]
n = len(data)

# 1. Point Estimate (Sample Mean)
x_bar = np.mean(data) # 100.4
print(f"Point Estimate: {x_bar}")

# 2. 95% Confidence Interval
# Using t-distribution because n < 30 and sigma unknown
# sem = Standard Error of Mean = s / sqrt(n)
standard_error = stats.sem(data)
ci = stats.t.interval(0.95, df=n-1, loc=x_bar, scale=standard_error)

print(f"95% CI: ({ci[0]:.2f}, {ci[1]:.2f})")
# Interpretation: We are 95% confident true mean is in this range.
