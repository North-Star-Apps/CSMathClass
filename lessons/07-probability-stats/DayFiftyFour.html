Section 1

Simple Linear Regression

Model: y = β₀ + β₁x + ε
y = response (dependent variable) x = predictor (independent variable) β₀ = intercept β₁ = slope ε = error term (random noise)
Section 2

Least Squares Estimation

MINIMIZE: Σ(yᵢ - ŷᵢ)² = Σ(yᵢ - β₀ - β₁xᵢ)² SOLUTIONS: β̂₁ = Σ(xᵢ - x̄)(yᵢ - ȳ) / Σ(xᵢ - x̄)² β̂₀ = ȳ - β̂₁x̄ Fitted values: ŷᵢ = β̂₀ + β̂₁xᵢ Residuals: eᵢ = yᵢ - ŷᵢ
Python Linear regression with sklearn 
from sklearn.linear_model import LinearRegression

model = LinearRegression()
model.fit(X, y)
print(model.intercept_, model.coef_)
Section 3

R² (Coefficient of Determination)

R² = 1 - (SS_res / SS_tot) = 1 - Σ(yᵢ - ŷᵢ)² / Σ(yᵢ - ȳ)² INTERPRETATION: • R² = 1: Perfect fit • R² = 0: Model explains nothing • R² = 0.8: 80% of variance explained
R² can be misleading! Always check residual plots.
