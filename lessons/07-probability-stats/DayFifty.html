Section 1

Random Variables

Random Variable X: Function X: Ω → ℝ mapping outcomes to numbers.
DISCRETE R.V. Countable values PMF: P(X = x) Σ P(X = x) = 1
CONTINUOUS R.V. Uncountable values PDF: f(x) ∫ f(x) dx = 1
Section 2

Expected Value

DISCRETE: E[X] = Σ x · P(X = x) CONTINUOUS: E[X] = ∫ x · f(x) dx KEY PROPERTIES: • E[aX + b] = a·E[X] + b • E[X + Y] = E[X] + E[Y] (always!) • E[XY] = E[X]E[Y] (if independent)
Linearity of expectation holds even for dependent variables!
Section 3

Variance & Standard Deviation

Var(X) = E[(X - μ)²] = E[X²] - (E[X])² σ = √Var(X) (standard deviation) KEY PROPERTIES: • Var(aX + b) = a²·Var(X) • Var(X + Y) = Var(X) + Var(Y) (if independent) • Var(X) ≥ 0
