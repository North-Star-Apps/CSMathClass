Section 1

Discrete Distributions

BERNOULLI(p): One trial, success prob p E[X] = p, Var(X) = p(1-p) BINOMIAL(n,p): n trials, count successes P(X=k) = C(n,k) p^k (1-p)^(n-k) E[X] = np, Var(X) = np(1-p) GEOMETRIC(p): Trials until first success P(X=k) = (1-p)^(k-1) p E[X] = 1/p POISSON(λ): Events in interval P(X=k) = λ^k e^(-λ) / k! E[X] = Var(X) = λ
Section 2

Continuous Distributions

UNIFORM(a,b): Equal probability on [a,b] f(x) = 1/(b-a) E[X] = (a+b)/2, Var(X) = (b-a)²/12 EXPONENTIAL(λ): Time between Poisson events f(x) = λe^(-λx), x ≥ 0 E[X] = 1/λ, Var(X) = 1/λ² NORMAL(μ,σ²): Bell curve f(x) = (1/σ√(2π)) e^(-(x-μ)²/(2σ²)) E[X] = μ, Var(X) = σ² STANDARD NORMAL: N(0,1) Z = (X - μ)/σ
Central Limit Theorem: Sum of many independent RVs → Normal!
