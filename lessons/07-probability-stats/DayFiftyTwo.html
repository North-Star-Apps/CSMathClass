Section 1

Markov's Inequality

For X ≥ 0: P(X ≥ a) ≤ E[X]/a
Example: If E[X] = 10, then P(X ≥ 50) ≤ 10/50 = 0.2 Very weak bound, requires only non-negativity!
Section 2

Chebyshev's Inequality

For any X: P(|X - μ| ≥ kσ) ≤ 1/k²
k = 2: P(|X - μ| ≥ 2σ) ≤ 1/4 = 25% k = 3: P(|X - μ| ≥ 3σ) ≤ 1/9 ≈ 11% Uses variance; tighter than Markov
Section 3

Chernoff Bounds

For sum of independent bounded RVs: Exponential tail bounds!
For X = Σ Xᵢ independent with μ = E[X]: P(X ≥ (1+δ)μ) ≤ exp(-δ²μ/(2+δ)) P(X ≤ (1-δ)μ) ≤ exp(-δ²μ/2) Much tighter than Chebyshev for sums!
Hoeffding: Similar exponential bounds for bounded RVs.
