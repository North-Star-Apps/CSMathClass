Dynamic Programming

Break problems into overlapping subproblems and store solutions to avoid redundant computation.

01

Core Concepts

DP applies when: 1. Optimal Substructure 2. Overlapping Subproblems
TWO APPROACHES: Top-Down (Memoization): Recursive + cache results Natural way to think about problems Bottom-Up (Tabulation): Iterative + fill table Often more space efficient
DP FRAMEWORK: 1. Define state: What info needed to solve subproblem? 2. Define recurrence: How do states relate? 3. Define base cases: Smallest subproblems 4. Compute order: Ensure dependencies resolved 5. Extract answer: Where is final solution?
02

Fibonacci Example

NAIVE RECURSIVE: O(2‚Åø) ‚Äî exponential F(n) = F(n-1) + F(n-2) WITH MEMOIZATION: O(n) ‚Äî linear Store computed values in cache BOTTOM-UP TABLE: O(n) time, O(n) space Fill table from F(0) to F(n) SPACE OPTIMIZED: O(n) time, O(1) space Only need last 2 values
// Top-Down with Memoization
function fibMemo(n, memo = {}) {
    if (n <= 1) return n;
    if (memo[n]) return memo[n];
    memo[n] = fibMemo(n-1, memo) + fibMemo(n-2, memo);
    return memo[n];
}

// Bottom-Up 
function fibTab(n) {
    if (n <= 1) return n;
    const dp = [0, 1];
    for (let i = 2; i <= n; i++) {
        dp[i] = dp[i-1] + dp[i-2];
    }
    return dp[n];
}

// Space Optimized
function fibOpt(n) {
    if (n <= 1) return n;
    let prev = 0, curr = 1;
    for (let i = 2; i <= n; i++) {
        [prev, curr] = [curr, prev + curr];
    }
    return curr;
}

Classic DP Problems

1D DP: ‚Ä¢ Climbing Stairs: dp[i] = dp[i-1] + dp[i-2] ‚Ä¢ House Robber: dp[i] = max(dp[i-1], dp[i-2] + nums[i]) ‚Ä¢ Coin Change: dp[i] = min(dp[i-coin] + 1) for all coins 2D DP: ‚Ä¢ Longest Common Subsequence (LCS) ‚Ä¢ Edit Distance (Levenshtein) ‚Ä¢ 0/1 Knapsack ‚Ä¢ Matrix Chain Multiplication
// 0/1 Knapsack
function knapsack(W, weights, values) {
    const n = weights.length;
    const dp = Array(n+1).fill(0).map(() => Array(W+1).fill(0));
    
    for (let i = 1; i <= n; i++) {
        for (let w = 1; w <= W; w++) {
            if (weights[i-1] <= w) {
                dp[i][w] = Math.max(
                    dp[i-1][w],  // don't take
                    dp[i-1][w - weights[i-1]] + values[i-1]  // take
                );
            } else {
                dp[i][w] = dp[i-1][w];
            }
        }
    }
    return dp[n][W];
}
