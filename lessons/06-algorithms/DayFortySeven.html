Section 2

Classic Examples

RANDOMIZED QUICKSORT: • Random pivot avoids worst case • Expected O(n log n) • Simple to implement RANDOMIZED SELECT: • Find kth smallest in O(n) expected HASH FUNCTIONS: • Universal hashing prevents adversarial inputs PRIMALITY TESTING: • Miller-Rabin: fast probabilistic test
Python Randomized QuickSort 
import random

def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = random.choice(arr)  # Random pivot!
    left = [x for x in arr if x < pivot]
    mid = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quicksort(left) + mid + quicksort(right)
Section 3

Analysis

KEY CONCEPTS: • Expected value: E[X] = Σ x·P(X=x) • Linearity of expectation: E[X+Y] = E[X] + E[Y] • Indicator random variables PROBABILITY BOUNDS: • Markov: P(X ≥ a) ≤ E[X]/a • Chebyshev: P(|X-μ| ≥ kσ) ≤ 1/k² • Chernoff: exponential concentration
