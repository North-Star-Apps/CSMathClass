Randomized QuickSort

Key insight: Random pivot selection avoids worst-case O(n¬≤) on adversarial inputs.
function quickSort(arr, lo = 0, hi = arr.length - 1) {
    if (lo >= hi) return arr;
    
    // Random pivot selection
    const pivotIdx = lo + Math.floor(Math.random() * (hi - lo + 1));
    [arr[pivotIdx], arr[hi]] = [arr[hi], arr[pivotIdx]];
    
    const pivot = arr[hi];
    let i = lo;
    for (let j = lo; j < hi; j++) {
        if (arr[j] < pivot) {
            [arr[i], arr[j]] = [arr[j], arr[i]];
            i++;
        }
    }
    [arr[i], arr[hi]] = [arr[hi], arr[i]];
    
    quickSort(arr, lo, i - 1);
    quickSort(arr, i + 1, hi);
    return arr;
}
Expected time: O(n log n) Worst case: O(n¬≤) but extremely unlikely With random pivot: any input behaves like average input
