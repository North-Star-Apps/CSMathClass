Section 1

The Greedy Strategy

Make the locally optimal choice at each step.

Example (Coin Change): To make 41 cents with fewest coins, take largest first: 25, 10, 5, 1. (Works for standard currency, fails for arbitrary systems).

Warning

Greedy doesn't always work (e.g., Knapsack 0/1 requires DP). But when it works (MST, Dijkstra), it's very efficient.

Section 2

Minimum Spanning Trees (MST)

Given a connected weighted graph, select edges to connect all nodes with minimum total weight.

  • Must be a Tree (No cycles).
  • Must span all vertices.
Section 3

Prim's vs Kruskal's

Prim's Algorithm

Node-based. Like Dijkstra. Start at a node, grow the tree by adding the cheapest edge connected to it.

Kruskal's Algorithm

Edge-based. Sort all edges by weight. Add edge if it doesn't form a cycle (using Union-Find).

Section 4

Huffman Coding

Lossless data compression.

Idea: Use short binary codes for frequent characters (e.g., 'e' = 0) and long codes for rare ones (e.g., 'z' = 110110).

Uses a Greedy strategy to build a binary tree.

Section 5

Python: Union-Find

Implementing Union-Find (Disjoint Set) for Kruskal's.

Python Disjoint Set 
class UnionFind:
    def __init__(self, n):
        self.parent = list(range(n))
        self.rank = [0] * n

    def find(self, i):
        if self.parent[i] != i:
            # Path Compression
            self.parent[i] = self.find(self.parent[i])
        return self.parent[i]

    def union(self, i, j):
        root_i = self.find(i)
        root_j = self.find(j)
        
        if root_i != root_j:
            # Union by Rank
            if self.rank[root_i] > self.rank[root_j]:
                self.parent[root_j] = root_i
            else:
                self.parent[root_i] = root_j
                if self.rank[root_i] == self.rank[root_j]:
                    self.rank[root_j] += 1
            return True # Connected successfully
        return False # Already connected

# Usage
uf = UnionFind(5)
uf.union(0, 1) # Connect 0 and 1
print(uf.find(0) == uf.find(1)) # True
