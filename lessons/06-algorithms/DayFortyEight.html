Section 1

Complexity Classes

P: Problems solvable in polynomial time Examples: sorting, shortest path, matching NP: Problems verifiable in polynomial time Examples: SAT, TSP, graph coloring P ⊆ NP (definitely) P = NP? (open problem, probably no)
P vs NP is one of the Millennium Prize Problems. $1 million prize!
Section 2

Polynomial Reductions

A ≤ₚ B: A reduces to B in polynomial time. If B is easy, so is A.
REDUCTION A ≤ₚ B: 1. Transform input of A to input of B (poly time) 2. Solve B 3. Transform output back (poly time) KEY INSIGHT: If A ≤ₚ B and B ∈ P, then A ∈ P If A ≤ₚ B and A is hard, then B is hard
Section 3

NP-Complete Problems

NP-COMPLETE: 1. Problem is in NP 2. Every NP problem reduces to it CLASSIC NP-COMPLETE PROBLEMS: • SAT (boolean satisfiability) • 3-SAT • Clique • Vertex Cover • Hamiltonian Path • Traveling Salesman • Graph Coloring • Subset Sum • Knapsack (decision version) COOK-LEVIN THEOREM: SAT is NP-complete (proved directly) All others proved by reduction from SAT
